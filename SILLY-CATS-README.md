Silly Cats — Bright & Colorful

A tiny static page that is super bright, colorful, and silly about cats.

Files added here:
- [index.html](index.html) — main page
- [styles.css](styles.css) — colors & animations
- [script.js](script.js) — spawn cats & confetti

Run locally:

```sh
# open directly (macOS)
open index.html

# or run a quick HTTP server
python3 -m http.server
# then visit http://localhost:8000
```

Click "Make it sillier!" or tap the playground to spawn cheerful cats and confetti.
